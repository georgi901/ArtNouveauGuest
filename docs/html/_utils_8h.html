Detailed Description

Funcții șablon utilitare pentru aplicația galeriei.

Acest fișier conține funcții șablon care pot lucra cu orice tip care are metodele necesare. Aceste funcții îndeplinesc cerința M3 pentru funcții șablon cu minim 2 instanțieri.
