Detailed Description

Excepție aruncată la erori de încărcare a datelor din fișiere.

Folosită când apar probleme la citirea sau parsarea fișierelor JSON. Poate indica eroare de deschidere fișier sau eroare de parsare.

std::ifstream fin(cale);
if (!fin.is_open()) {
throw IncarcareDataException(cale, "fisier");
}
