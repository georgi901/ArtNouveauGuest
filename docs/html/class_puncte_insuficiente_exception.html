Detailed Description

Excepție aruncată când utilizatorul nu are suficiente puncte.

Folosită când se încearcă o achiziție cu puncte insuficiente. Oferă informații despre punctele disponibile și cele necesare.

if (puncte < cost) {
throw PuncteInsuficienteException(puncte, cost);
}
