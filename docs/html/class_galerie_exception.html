Detailed Description

Clasă de bază pentru toate excepțiile din aplicația galeriei.

Derivată din std::exception, această clasă oferă un mecanism uniform pentru tratarea erorilor în întreaga aplicație.

try {
// operație care poate eșua
} catch (const GalerieException& e) {
std::cerr << "Eroare galerie: " << e.what() << std::endl;
}
